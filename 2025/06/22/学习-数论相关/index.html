    Stern-Brocot树是一种可以不重不漏列举有理数的方式,它的构造如下:

    一开始,序列中有两个分数:$ { 1 } \(和\) { 0 } $,这里使用了$0 $作分母,但我们暂且认为它是正确的,因为这样会出现很多方便的性质.

    我们不妨认为$ { 0 } = + $,那么不难发现这么构造序列,所得到的序列一定是单调递增的.
